Ejercicio Tropiko

Descripción: Este proyecto busca recrear una plantilla web aplicando conceptos de HTML y CSS, enfocándose en la maquetación, estilos y uso de imágenes.
Fue desarrollado como ejercicio académico para reforzar conocimientos en diseño web y organización de código.

Características:
- Estructura de la página en HTML.
- Estilos personalizados con CSS.
- Integración de imágenes para enriquecer el diseño.
- Diseño adaptable y organizado en carpetas.
  
Tecnologías utilizadas
- HTML5
- CSS3
  
Estructura del proyecto:

images/       # Carpeta con recursos gráficos

index.html    # Página principal

styles.css    # Estilos de la página
